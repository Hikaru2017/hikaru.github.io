没有在深夜痛哭过的人不足以谈人生

Posted on 2019-06-29

“Why do we say, I still love you, but I do not like you anymore?”
老生常谈的话题了，但我却想抽出五分钟的时间重新聊一聊，也顺便记录下自己此时此刻的心情。
焦虑，不安却又效率低下无所事事，一边心惊胆战的浪费着时间，一边又朝着那些看似努力的人吐口水，这就是我目前的状态与心情。
毕业在即，我却像大梦初醒般的明白了很多事，比如我还爱你，比如我喜欢tk，比如那些巧妙的模板邮件？又比如我竟浪费了那么多时间。
前几日接到了M的电话，随意说了几句却不欢而散，挂了电话后我开始反思自己这样的自己我厌恶的情绪要到什么时候？
如今已经不是能不能接受约与结婚的这样层面的问题了，如今我是连听到聊骚都从心底里开始恶心。
这样的日子要到什么时候啊？
这样的日子要到什么时候啊？我不停的问自己。
这样的日子要到什么时候啊？
怕是还要三五年呢。

窗外的云像是棉花糖

Posted on 2018-09-25 | In Diary

//第一次写这篇儿的时间已经忘记了 暂且记为今天
        从大阪回来也一周多了，一直想找个时间记录一下这短暂的关西游却总是找不到合适的机会，也罢，不如就现在开始写。
        今天是我到学校的第354天了，也就是说再有11天就是一年整，在这将近一年的时间里我还是第一次跨出学校所在的xx县的大门，去“大城市”体验了一把，有喜有悲吧。先说说喜的方面，看到了很多不一样的东西。悲就是我越来越意识到只有到大的地方才能见识到不一样的世界，不一样的人，不一样的价值观，不一样的生活方式，也是第一次见到了传说中的躺在大街上睡觉的人。
        走走停停，停停走走，虽说是借着考试的由头出去逛了逛，但总觉得还是缺点儿什么，我时常会想，要是我没来现在的学校而是选择去读了语言学校，估计现在也还在准备着入学考试吧？能开口说日语了吗？考过考试了吗？
        窗外的云像是棉花糖，我已经有点儿记不起来是在什么心情下写下的这句话，应该是坐在座位上，一边缅怀着时间的飞速流逝，一边又肆无忌惮的浪费着时间抬起头看着窗外发呆吧。我最近时常会问我自己一个问题，如果我当初选择了另一条路那现在又会是什么样呢，是节衣缩食住在合租房每天挤着两三个小时的地铁上下班搬砖，又或是被人情世故拖拉着消耗着想要挣脱但是却找不到退路，又或是会后悔当初该选择出国而不是工作？

Posted on 2018-08-05

        最近开始重新学习并使用kali linux，其实早在两三年前我就了解并学习过一些关于kali的基础性的知识，现在重新捡起来也不算晚吧。现在记录一下我在安装并使用kali的过程中遇到的一些问题以及安装完成后的一些配置工作。

  1. unable to locate package linux-headers 不能定位头文件的软件包
    解决办法，先更新系统再安装:
    sudo apt-get update && apt-get upgrade && apt-get dist-upgrade</code>
    apt-get install linux-headers-$(uname -r)
  2. 安装virtualbox的增强功能失败
    首先从光盘中拷贝出VBoxLinuxAdditions.run文件并赋予执行权限chmod +x VBoxLinuxAdditions.run，然后运行。
