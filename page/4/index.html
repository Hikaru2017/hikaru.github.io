感情论存在的价值

Posted on 2018-01-22

        忽然间思绪万千，想记下些什么，细细想来，距上一次写字儿已然半年有余，今天的话题是，感情论存在的价值。
        坐在同样的教室里，相似的位置，不同的时间与人物，当然还有心情与心境，长久以来，我都以御姐控女王控自诩，不能否认的是，青春期时期的一些小说与影视节目间接的潜移默化的影响了我的感情观，或许也与我的原生家庭有一定的关系？不得而知。又或许骨子里我就是一个好吃懒惰且懦弱的人呢，希望有强大的人从天而降，希望被保护被照顾。到底是什么激发了我写下这些字的冲动呢？是因为感慨时间流逝自己却毫无长进？还是因为感情过渡到了新的阶段需要回首往事找找存在感？或许只是不想听课打发时间？
        日常的夸夸其谈
        总的来说，我是个懦弱且善于自嘲的人，与此同时还是个戏精。所谓戏精，就是总是期盼/幻想那些不可能发生的事情的人，比如觉得全世界的年长女性都喜欢我，但事实上，我又怀疑着是不是这个世界上根本不会有人喜欢我。我始终不相信日久生情这样的例子，于我而言，第一次见面就能给这个人划定合适的圈子，只会随着时间的推移与日常生活的接触向下兼容，至今还未曾发生过‘我原本不喜欢这个人（对这个人无感）， 但接触了之后突然产生了好感’类似于这样的例子。很矛盾的是，我又常常会在某一瞬间被某个点所击中，进而演变成我喜欢这个人的这一点，我喜欢这个人，到底是日久生情呢还是原本对这个人印象就不错，被某个点激发后，顺其自然的升华了感情？不得而知。
        是从什么时候开始的喜欢
        我无法回答这个问题，同样的我也不能找到一个准确的时间点或时间段，当我发现我喜欢你的那一刻，事实上我已经走了一半的路。长久以来，我都以“我想”为基准，一旦我产生了“我想见到这个人”这样的想法，耳边就会有“完了，我可能喜欢这个人”这样的声音。但往往又会因为自己的自卑或是自傲不愿承认，抓住对方的一个点无限的放大，用“我怎么会喜欢这样的人呢”来劝诫自己，经过数百个回合的纠结与自我劝诫，最后不了了之。
        对“强大的”女性的看法
        其实这点没什么好争议的，我喜欢这样的人且一心先要成为这样的人，哪怕我不能成为这样的人，我也在努力的走到这样的圈子中。
        性与爱能分开吗
        理论上我觉得可以，但事实上，于我而言，并不能，似乎是某种近乎变态的洁癖让我不能接受无感的人的触碰。
        未来的打算
        无。不过倒是很想尝试一下之前没有的经历，在自己可以接受的前提下。
        未完待续。

Ubuntu下更新sublime text 3

Posted on 2017-10-18

Install the GPG key:
wget -qO – https://download.sublimetext.com/sublimehq-pub.gpg | sudo apt-key add –
Ensure apt is set up to work with https sources:
sudo apt-get install apt-transport-https
Select the channel to use:
Stable
echo “deb https://download.sublimetext.com/ apt/stable/” | sudo tee /etc/apt/sources.list.d/sublime-text.list
Dev
echo “deb https://download.sublimetext.com/ apt/dev/” | sudo tee /etc/apt/sources.list.d/sublime-text.list
Update apt sources and install Sublime Text
sudo apt-get update
sudo apt-get install sublime-text

那些日子你会不会舍不得

Posted on 2017-07-21

        题目源于歌曲《原谅》，原唱为张玉华，偶然听到了刘瑞琦的翻唱版本，个人认为比起张玉华清脆的嗓音多了一丝丝的沧桑。十六七岁我特别叛逆的时候，喜欢这首歌，那个时候更多的是在表达喜欢而不得、离别却不舍等等情感，如今再细细品味时，竟读出了一分亲情。
        前两天去找我妈拿东西，天气特别热，大概有近40度，距离小区1.5公里多的地方在修路，只好把车停在路边，顶着明晃晃的大太阳走了十几分钟才找到她的底店。一路上我后悔了无数次应该拿把伞再出门，汗水从脸上开始滑落，混合着我的眼泪护肤品化妆品防晒霜一起浸湿了黑T恤。
        七拐八绕的好不容易找到了她住的地方，原本想着拿上我要取的东西就立马走，但她很热情的邀请我进去坐坐，还给我倒了两杯热水(虽然我从来不喝热水)，我盯着桌子上的热水杯发呆了两秒钟，想着坐会儿就坐会儿，开始说一些客套话。她回厨房拿了一个空碗，把杯子里的热水倒进碗里，再把碗里的水倒回杯里，来回倒腾，让水能凉的快一点。有那么一瞬间我觉得自己不是个“大人”了，我是这个人的女儿，受到她的呵护与关爱。你以为我这些波涛汹涌的心理活动会给我和她之后的相处或是生活带来什么改变吗？不会的，一点改变都没有。
        聊了大概两个小时，我说我要走了，回头再联系她，她坚持要送我出小区，我说不用了，我能找的到。她反复说了三四次要送我，我回头望着她有些恳求却又期盼的脸，迎风飞舞着的白发，定是一两个月没染过头发了，还有些佝偻的背。是啊，她已经不再年轻了。“想送就送吧，走吧，拿好钥匙，把门锁上。”我这样说道。其实我只是觉得送来送去的麻烦，我也不太能理解想和一个人多待几分钟几秒钟的心情。说到这想起来一个段子，我一直自喻自己是个自立的人，自己的事情自己做，很少很少很少去麻烦别人，在我看来自己觉得麻烦的事情别人也会觉得麻烦呀，比如说把重的东西分给别人，我每次会觉得这个行为很奇怪，你自己拿着觉得重，你分给别人，别人就不觉得重了吗？也许是这些奇奇怪怪的想法让我看起来有些冷漠不近人情，其实不然，在我看来，不轻易打扰别人、不给别人添麻烦应该是成年人之间心口相传的秘籍。
        有时候我会突然好奇一些俗事，就如歌中唱到的“那些日子你会不会舍不得” “你都如何回忆我 带着笑或是很沉默”。其实我有很多很多舍不得的人或东西，但大部分都被我刻意的从的记忆里删除了。通常我舍不得的point和别人不太一样，比如住了六七年突然要搬家了，我最舍不得是门口菜市场里一个卖菜的老奶奶，因为我搬走了就意味着又少了一个像我这样特意去她家买菜的人。几年前从初中的家搬到高中学校对面的家，最舍不得的是初中家门口洗澡堂的一位搓澡阿姨，可惜了，后来这么多年再没遇到那么好的搓澡阿姨。
        最近在读余华的《现实的一种》，其中有一段甚是喜欢，摘出来单独说一下：“回忆在岁月消失后出现，如同一根稻草漂浮到溺水者面前，自我的拯救仅仅只是象征。同样的道理，回忆无法还原过去的生活，它只是偶然提醒我们：过去曾经拥有过什么？这样的提醒时常以篡改为荣，不过人们也需偷梁换柱的回忆来满足内心的虚荣，使过去的人生变得丰富和饱满。”
        为什么回忆总是美好的，因为可以随便篡改呀，以当前的思想和行为去回首往事是一件特别愚蠢的事情，哎，回首往事本就是一件愚蠢的事情呀。:-p
