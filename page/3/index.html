Ubuntu root的默认密码

Posted on 2018-05-15

安装完Ubuntu之后切换root用户，发现需要输入root的密码，然而我并不知道root的密码是什么，此时需要使用当前账户对root的密码进行重置: sudo passwd root，重新设置密码后即可切换root用户，su root。
The root account is disabled when you first install Ubuntu. The first user created during the installation has administrative rights on the system, and can run programs as root with sudo, using only their normal user password. For example: sudo apt-get update.
If you wish to use the root account in more traditional UNIX fashion, you can set the root password by typing sudo passwd root. This will allow you to use su or login as root on the console.
If you need a shell with root privileges, run sudo -s.
All uses of sudo will require the user’s password.

迟钝还是假装

Posted on 2018-03-22

        有时候觉得自己特别迟钝，像是那种特意的、故意的、假装出来的自己很迟钝，进而演变成真的很迟钝。
        比如前两天说开会的时候，大家轮流汇报进展，一phd说：‘我注册这个网站等了一天多他们才审核完通过申请(言外之意他们太慢)’，教授说你得考虑到不在一个时区，他们在欧洲。phd又说我给他们负责人发了三封邮件，等了两天多了，到现在还没回。教授抬起头说，别逼他们这么紧，发一封就行了，如果你发三封邮件，they will hate you。不知道为什么，他们会讨厌你这句话我听了就觉得特别好笑，虽然坐在教授对面，但我实在是忍不住抿着嘴笑了好长时间，心里想着不行，我一会儿得和你说说这个差点儿没把我笑趴下的事儿。
        开完会坐在工位上，突然想到，啊，现在的关系我已经不能和你分享这件事情了。
        很久很久之前，久到我已经记不起来到底是什么时候了，大约过了半个世纪那么久，我曾以为爱一人最深的境界就是：‘我只有一碗饭，你和我都很饿，我宁愿自己挨饿，也会让出多半碗甚至一碗来给你吃’。到现在我依旧这么认为，或许将来的某一天某个人的出现会改变我这一观点呢，期待着。现在可以做的是，再这条认知前面再加上一条，喜欢一个人最深的境界是什么呢？是听到让人觉得开心的事第一反应就是，哈哈哈，这个好搞笑，赶紧记下来回去和你说，这份期待与分享的心情太过强烈，以至于让我忘记了几个小时前才吵过一架。
